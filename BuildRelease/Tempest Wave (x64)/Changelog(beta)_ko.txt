// BETA changelog of 2.1.0


/ 2.1.0b2 (release date: 18.06.29)

+ 6개의 테마를 추가했습니다.
+ PLATINUM 게임 모드 관련 업적을 추가했습니다.
= PLATINUM 게임 모드의 판정 기준을 조정했습니다.
= Creator에서 구분선 선택 순서를 조정했습니다.
= 선곡 화면에서 간혹 정렬이 정상적이지 않던 문제를 수정했습니다.
  = 프로그램 내에서 따로 정렬하므로, 기존의 정렬 순서와는 차이가 있을 수 있습니다.
= Creator에서 채보를 재생할 때 노트 소리가 간혹 잘못 출력되던 문제를 수정했습니다.

/ 2.1.0b1 (release date: 18.06.21)

+ Unity를 2018.1로 업데이트했습니다.
  + .NET 4.x 스크립팅 런타임을 적용했습니다.
  + Android 버전에 IL2CPP와 64비트 지원을 적용했습니다.
+ PLATINUM 게임 모드를 추가했습니다.
  + 1L PLATINUM 모드를 플레이할 수 있습니다.
  + 이에 따라, Creator에서 1라인 채보를 만들 수 있습니다 (tw1).
  = PLATINUM 게임 모드는 아직 개발 중이므로, 현재로서는 완벽하지 않습니다.
+ 테마 선택 기능을 추가했습니다.
  + 지금은 5개의 테마가 있습니다.
  = 더 많은 테마가 추후에 추가될 예정입니다.
